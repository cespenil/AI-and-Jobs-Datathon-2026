# AI-and-Jobs-Datathon-2026

Contributors: Chloe Nicole Espenilla, Varsshine Rajesh, Jeanna Park, Shamikta Suresh Raj

Inspiration and Dataset

We were inspired by how AI is transforming careers and impacting salaries. Our goal was to identify which jobs and skills are most resilient to AI automation and how AI risk affects job prospects.
We used a free dataset from Kaggle: https://www.kaggle.com/datasets/shree0910/ai-job-risk-and-salary-dataset-20152035

Code Overview

Data Cleaning & Feature Engineering
-> Aggregated job-level data from the dataset.
-> Created several features based on the variables provided by Kaggle's dataset for visualization and analysis.
Machine Learning Analysis
-> Built a linear regression model using AI risk, skill demand, and salary as features to predict risk-adjusted openings.
Visualizations
-> Created charts with matplotlib from the Machine Learning Analysis.
-> Compatible with Omni's visualization tool.
